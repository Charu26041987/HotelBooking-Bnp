 ## Booking
## Hotel Booking API Automation Framework

 ## Overview
This project is an API test automation framework for a Hotel Booking system.

It is built using **Java, Maven, Cucumber (BDD), and Rest Assured** to validate booking and authentication APIs.

---

##  Tech Stack
- Java 17
- Maven
- Cucumber (BDD)
- Rest Assured
- TestNG / JUnit

---

##  Project Structure
src/test/java
- runner → Test runner class  
- stepdefinitions → Step definition files  

src/test/resources
- features → Cucumber feature files
